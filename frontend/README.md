# MyOwn Proto / PNC Hub

Проект, созданный для того, чтоб в одном месте собрать ряд моих инструментов и разработать новые.

## Что внутри
- Мониторинг health/performance стендов
- Статистика и графики
- Планирование тестов
- Удалённое управление (Remote control)
- Debug консоль

## Стек (frontend)
- React 19 + TypeScript
- Vite
- Tailwind CSS + Shadcn/UI
- Lucide icons
- React Router

## Запуск
```bash
cd frontend
npm install
npm run dev
